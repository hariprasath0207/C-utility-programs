# ATM Simulation Utility

An interactive command-line utility simulating basic banking operations for two distinct users. It handles user authentication, balance updates, deposits, secure withdrawals, and peer-to-peer transfers.

## 🚀 Key Features
* **Dual-User Profiles:** Supports separate login profiles with unique credentials.
* **Authentication PINs:** Simple, password-protected login flows.
* **Core Functions:**
  * **Balance Inquiry:** Real-time account balance updates.
  * **Cash Withdrawal:** Safe withdrawals with insufficient balance checks.
  * **Cash Deposit:** Seamlessly deposit money into the active account.
  * **Account Transfer:** Move funds securely between the two profiles.
